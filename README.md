# IvyOS Installer
IvyOS Installer is designed to be ran on a freshly installed copy of Arch Linux to install the packages!

IvyOS Installer is just like the ML4W installer, other than it enhances the entire system,
not just the "UI Framework" (that is how techinally inclined Android users call DEs in Android.)

Review the `ivyos-install.sh` file to see what it actually does!
This is a bash script, that will install everything that makes IvyOS, well IvyOS!

Read the `README_.md` for more information about the distro!
